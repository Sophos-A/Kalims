Common Issues

  • Failed to fetch - This usually indicates a network connectivity issue or CORS problem
  • CORS error - Check if the backend ALLOWED_ORIGINS includes the frontend origin
  • 404 Not Found - The API endpoint doesn't exist or the URL is incorrect
  • 500 Server Error - There's an issue with the backend server
